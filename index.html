Introduction

HTML is an acronym for Hyper Text Markup Language and it is the standard language for creating web-pages. Every webpage you surf on your browser is built using HTML. The primary use of HTML to provide a proper structure to the web page so all the content or data of the page could represent adequately.A stand-alone HTML can create only stastic and skeleton looking black and white pages, but with the help of CSS and JavaScript, we can create a more interactive and inuitive webpage. When we try to visit a website or click on the link, we basically request the server to show us the page, then the server acts on our request and sends us an appropriate HTML document as a response. Then this HTML document parsed by browse, and we are able to see the content.

HTML Elements

The HTML elements provide the semantic meaning to to the web-page content. We usually interchangeably use the term HTML element and tags, but technically both are different. An HTML tag is just a character inside the angle bracket < >, whereas the HTML elements is a collection of starting tag, its attribute, content and end tag. For example:

<p class= "para"> Hello World </p>

HTML Attributes

In HTML, the attributes are used to provide additional information about the elements. For most of the HTML elements, attributes are optional, but there are some elements where we have to deliver the attributes. Attributes always specified within the opening tag of the HTML element and they specified in a name and value pair.Example

<image src= "cat.jpg" alt ="cat image">

There should be at least one space gap between two attributes. The value must reside in the double inverted comma.

HTML Heading

To display the section heading, title or subtitle we use the HTML heading tags. In HTML 5 we have 6 heading tags start from <h1> up to <h6>, where <h1> specify the largest heading and <h6> represents the smallest or subheading. If the content is specialized br heading tags, then it would be displayed large and bold as compared to other text content present on the web-page.Example

<h1>First Heading </h1>
<h2>Second Heading </h2>
<h3>Third Heading </h3>
<h4>Forth Heading </h4>
<h5>FIfth Heading </h5>
<h6>Sixth Heading </h6>

HTML Formatting

In HTML, we have many special elements that can provide meaning to text content.

HTML Elements Description
<b> Bold the text
<strong> An alternative for <b>, which tell that this text is essential.
<i> Italic the text
<em> Similar to italic but used when we want to emphasize the text.
<mark> It marks the text with a default "yellow" background colour
<small> Decrease the text size
<del> It prints a cross line over the text.
<ins> Represents the inserted text by putting an underline.
<sub> This element is used to dispaly the subscript
<sup> It can make a text superscript.
