HTML DOCUMENTATION

Introduction

HTML is an acronym for Hyper Text Markup Language and it is the standard language for creating web-pages. Every webpage you surf on your browser is built using HTML. The primary use of HTML to provide a proper structure to the web page so all the content or data of the page could represent adequately.

A stand-alone HTML can create only stastic and skeleton looking black and white pages, but with the help of CSS and JavaScript, we can create a more interactive and inuitive webpage. When we try to visit a website or click on the link, we basically request the server to show us the page, then the server acts on our request and sends us an appropriate HTML document as a response. Then this HTML document parsed by browse, and we are able to see the content.

HTML Elements

It provides a semantic meaning on the web content. We interchangeably use HTML elements and Tags. HTML tag is a character inside the angle brackets <>. HTML element is a collection of starting tag, attributes, content and end tag.

HTML Attributes

It is used to provide additional information about the elements. Attributes always specifies the name and value which are placed within the opening tag of the HTML element. There should be at least one space gap between two attributes. The value must reside in the double inverted comma.

HTML Heading

We use HTML headings tags for heading, title and subtitle

There are 6 heading tags where h1 specifies the largest font while h6 specifies the smallest font header

Html Paragraph

A paragraph in HTML is defined by element p in tags. It shows starting a new line.

HTML Style

This displays a default style for HTML document.

HTML Formatting

There are several special elements that gives a special meaning to text context. They include Bold, Strike, Underline, Delete,Large, Small, Strong, Italics, Mark, Small, Insert, Subscript, Superscript and Empasis.. They should be in their respective tags.
